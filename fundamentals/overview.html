Overview

DevNet System library is the fundamental infrastructure of DevNet Framework, which includes the following components:

  • Generic Collection
  • Evnet Delegate
  • Async Task
  • Extension Method
  • Linq Method
  • Database Connector
  • Generic Compiler

Requirements

  • PHP 7.4 or higher version
  • Composer Dependency Manager

installation

If you already installed DevNet CLI or created an DevNet Project, you have already the System Library included in your project, but you can install DevNet System as a third-party library for any framework project, by running the following command in the terminal: 


composer require devnet/system
